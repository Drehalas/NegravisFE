POST/api/oracle-manager/query

Oracle Query

Execute queries across multiple Oracle providers with consensus algorithms

Parameters

providerrequiredstring

Oracle provider (chainlink, coingecko, weather, nasa, wikipedia)

queryrequiredstring

Query string or symbol

userIdoptionalstring

User identifier for tracking

Example Request

Terminal
curl -X POST "https://negravis-app.vercel.app/api/oracle-manager/query" \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "chainlink",
    "query": "BTC",
    "userId": "user123"
  }'
